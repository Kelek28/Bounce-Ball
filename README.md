### Built With

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Kelek28/Bounce-Ball.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run it
  ```sh
  npm run serve
  ```
 Enjoy
